--Food Delivery Analytics--
Project Overview

This project analyzes food delivery data to understand customer preferences, restaurant performance, delivery time, and ratings. The goal of this project is to extract useful insights from the data and visualize them using an interactive dashboard.

--Tools Used

Excel – Data cleaning and preprocessing

MySQL – Data storage and analysis using SQL queries

Power BI – Data visualization and dashboard creation

--Dataset

The dataset contains information about:

Restaurant name

City

Cuisine type

Order ID

Delivery time

Customer rating

Order date and year

Project Workflow

Cleaned and prepared the dataset using Excel.

Imported the dataset into MySQL database.

Performed data analysis using SQL queries.

Connected the processed data to Power BI.

Built an interactive dashboard to visualize insights.

Dashboard Features

--The Power BI dashboard includes:

Average Rating of restaurants

Average Delivery Time

Top Cuisine Type based on orders

Restaurant Count by Rating

Restaurant Count by City

Delivery Time vs Rating Analysis

Rating Trend by Year

--Users can filter the dashboard using:

Month

City

Cuisine type

Key Insights

Chinese cuisine has a high number of orders compared to other cuisines.

Cities like Bangalore, Chennai, and Delhi have a large number of restaurants.

Delivery time influences customer ratings.

Ratings show changes across different years.
